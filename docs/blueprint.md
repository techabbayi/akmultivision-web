# **App Name**: AKMultiVision Digital

## Core Features:

- Hero Section: Displays the agency name, tagline, and call-to-action buttons.
- Services Overview: Displays the services offered by the agency, like web design, graphic design and social media and ad management.
- Pricing Plans: Showcases the pricing plans and their included features, such as Starter, Pro, and Advance plans.
- Portfolio Showcase: Presents a portfolio grid with sample projects, poster previews, and ad banners to demonstrate the agency's capabilities.
- Social Media Branding: Displays mockups of social media branding on platforms like LinkedIn, Instagram, and Twitter (X).
- Contact Form: Collects user information (name, email, phone, message) and stores it in Firestore.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to reflect sophistication and digital innovation.
- Background color: Very dark purple (#1A1A2E), offering a modern and sophisticated base.
- Accent color: Bright blue-violet (#8A2BE2) as a vibrant contrast, attracting user attention.
- Body font: 'Inter', a sans-serif font known for its readability and modern appearance, used for main text and descriptions.
- Headline font: 'Poppins', a geometric sans-serif used to create a fashionable, avant-garde look.
- Utilize modern and clean icons from Lucide-react to enhance visual communication.
- Implement a responsive layout optimized for various devices (mobile, tablet, desktop) to ensure accessibility.
- Use subtle Framer Motion animations to add engaging visual elements and a smooth user experience.
